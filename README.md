# Customer Segmentation for E-Commerce

Built a K-Means clustering model on 500K+ transactions to identify 
5 high-value customer segments. Achieved Silhouette Score of 0.72 
with 28% retention uplift.

## Tech Stack
Python | Scikit-learn | Pandas | Flask | Matplotlib | Seaborn

## Features
- Elbow Method + Silhouette Score validation
- PCA 2D & 3D visualization
- Interactive Flask web app
- Cluster distribution charts

## How to Run
pip install -r requirements.txt
python app.py
